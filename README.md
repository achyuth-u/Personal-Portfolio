##  Key Features

- **2025 Bento Grid Architecture:** A modern, asymmetrical layout inspired by high-end tech products.
- **Dynamic Theme Engine:** Smooth transitions between Dark (Deep Black) and Light (Soft Grey) modes.
- **Visual Engineering Branding:** Custom-built typing effect cycles through core professional identities.
- **Navigable Works Gallery:** A masonry-style gallery for graphic design projects with a navigable full-screen lightbox.
- **Interactive Credentials:** Accordion-style bento tiles for professional experience, education, and certifications.
- **Performance First:** Lightweight architecture using vanilla JS and CSS variables (no heavy frameworks).
- **Adaptive Favicon:** An SVG-based favicon that automatically toggles its color based on the user's system theme.

##  Tech Stack

- **Structure:** [HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)
- **Styling:** [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS) (Variables, Flexbox, Grid, Keyframe Animations)
- **Logic:** [JavaScript (ES6+)](https://developer.mozilla.org/en-US/docs/Web/JavaScript)
- **Icons:** [Lucide Icons](https://lucide.dev/)
- **Typography:** [Sora](https://fonts.google.com/specimen/Sora) (Headings) & [Inter](https://fonts.google.com/specimen/Inter) (Body)
